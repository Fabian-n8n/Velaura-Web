# Nodemation Web

> AI automation agency website — dark, cinematic, premium.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Tech Stack

- **Vite 5** + **React 18**
- **Tailwind CSS v3**
- **framer-motion** — BlurIn + SplitText animations
- **hls.js** — HLS video streaming (Mux)
- **lucide-react** — icons

## Build

```bash
npm run build    # outputs to /dist
npm run preview  # preview production build
```
